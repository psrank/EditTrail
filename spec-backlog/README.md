# EditTrail OpenSpec Package

EditTrail is a JetBrains plugin for IntelliJ IDEA and Rider that provides a visual, searchable history of recently viewed and edited files.

This package contains iteration-based OpenSpec proposal documents.

## Product Summary

EditTrail helps developers navigate large codebases by showing the files they have recently viewed or edited, enriched with search, file-type filters, and visual grouping.

The plugin is intended to reduce reliance on the project tree when working across multiple modules, packages, folders, or layers.
